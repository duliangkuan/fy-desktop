⚠️ 检测到网络连接被拦截，启动 CC 可能失败

你可能开着 VPN / 代理软件（Clash 等），它把 EasyCC 到接入服务商的连接劫持了。两种解决办法（任选其一）：

办法一（最简单）：彻底退出 VPN——右下角托盘图标右键退出整个程序（仅关闭窗口无效），然后点下方「重新检测」。

办法二（VPN 不想退）：给代理软件加直连规则，把你所用服务商的域名（如 api.deepseek.com）设为 DIRECT 直连，保存重载。

欢迎使用 EasyCC 👋

三分钟用上 Claude Code——用你自己的国产大模型 API，不用翻墙、不用海外订阅。

1
选服务商 · 填 Key

「设置 · 接入」的模型广场挑一家（推荐 DeepSeek），粘贴你自己的 API Key

2
点启动

回启动页点「▶ 启动」，首次自动下载运行组件（仅一次）

3
开始写代码

在弹出的终端里对话式写代码、改项目、跑任务
